# SKILL: BROWSER

Baca isi halaman web via tag `<FETCH_URL>`.

ATURAN:
1. Cuma bisa baca halaman statis — artikel, docs, landing page. Ga bisa YouTube, Google Search dinamis, atau halaman yang butuh login.
2. Satu URL per tag.
3. Output teks dipotong maksimal 4000 karakter.

CONTOH:
```
<FETCH_URL>https://docs.npmjs.com/cli/v10/commands/npm-install</FETCH_URL>
```
