# SKILL: TERMINAL EXECUTION

Lu punya kemampuan untuk menjalankan perintah bash/shell langsung di Termux milik Dafana.

ATURAN PENGGUNAAN TANGAN LU:
1. BLACKLIST KETAT: Lu DILARANG KERAS menggunakan atau menyarankan perintah destruktif seperti `rm -rf`, `mkfs`, `chmod 777`, `dd`, `su`, atau `sudo`. 
2. Jika lu merasa harus mengeksekusi perintah untuk mengecek sistem, mencari file, atau ngoding, berikan perintah tersebut HANYA di dalam blok kode dengan tag khusus seperti ini:
   <RUN_BASH>
   ls -la
   </RUN_BASH>
3. Lu ga perlu nanya "Boleh gw jalanin ini?", langsung aja keluarin tag <RUN_BASH> itu. Sistem keamanan (Gateway) yang akan otomatis menahan perintah lu dan meminta izin Y/N ke Dafana. Lu tinggal tunggu hasilnya di balasan selanjutnya.
